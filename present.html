02 · Orchestration des Agents

Pipeline Étape par Étape

Le pipeline s'exécute de manière déterministe pour chaque module généré. Cliquez sur une étape pour l'inspecter.

1
Réception

Requête API Ingestée

Validation JWT et chargement du contexte du plan d'apprentissage depuis PostgreSQL.

2
Mémoire RAG

Recherche Sémantique pgvector

Récupération des leçons précédentes similaires via cosinus-similarité pour assurer la cohérence.

3
Researcher · Groq

Appels d'Outils en Parallèle

Groq interroge Wikipédia, Google et YouTube simultanément pour obtenir des sources vérifiées en ~250ms.

4
Writer · Gemini

Génération de Contenu Profond

Gemini 2.0 Flash synthétise les recherches pour rédiger un module structuré (Markdown + Schémas).

5
Critic · Hybrid

Validation gate + jsonrepair

JSON structure parsed. jsonrepair corrects malformed outputs deterministically. Mermaid syntax validated. Pedagogical score computed. If score < threshold → Writer revision loop.

6
Persistance

Écriture PostgreSQL + pgvector

Sauvegarde du module et indexation vectorielle immédiate pour la mémoire à long terme.

7
Livraison

Module délivré au Client

Le module structuré est affiché dynamiquement avec des schémas Mermaid et des quiz interactifs.
